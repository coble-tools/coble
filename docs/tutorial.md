# Using COBLE to replicate given environments

Replicating environments from manuscripts or github sites is never as easy as you might hope. Various errors occur from operating system, to the advance of time and archiving of channels, compile incompatibility and dependency errors. Multiple package managers mean resolution of versions across package managers is not done leaving manual intervention necessary that may be bespoke to user, machine and environment. These tutorials take you through taking a specification from a publication to build an environment, and troubleshooting.

## SYLVER - finding the packages
SYLVER is a publication from the ICR with a coherent description of the code in the [Code availability section](
https://www.nature.com/articles/s41588-025-02108-2#code-availability). The [tutorial](tutorials/sylver.md) takes you through finding where those packages are and building the recipe.


## CountReg - troubleshooting errors
CountReg throws some errors on install and this [tutorial](tutorials/countreg.md) takes you though analysing the errors and fixing the recipe input.
